# Workshop Update Overview (BKK 2026)

This directory contains updated instructions and supplemental information for the ["App Mod Workshop"](https://codelabs.developers.google.com/codelabs/app-mod-workshop) workshop. These changes reflect recent updates to the Google Cloud Console UI and the integration of new AI capabilities.

## Supplemental Instructions

- [Module 1: Create a SQL Instance](./module-01.md) - Instructions for the new Cloud SQL landing page and regional settings.
- [Module 3: Deploy the App to Cloud Run](./module-03.md) - Critical IAM permission fixes for the default compute service account.
- [Module 7: Empower your App with Google Gemini](./module-07.md) - Migrating to Gemini 2.5 Flash and using the revamped Cloud Run Functions UI.
